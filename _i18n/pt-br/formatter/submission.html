Como enviar as informações da sua conferência
1. Formato CSV do Python-Organizers

Adicione sua conferência ao repositório python-organizers/conferences:

  • Faça um fork do repositório
  • Edite o arquivo CSV do ano correspondente
  • Adicione a linha de dados da sua conferência
  • Envie um pull request
2. Formato YAML do PythonDeadlin.es

Adicione sua conferência ao jesperdramsch/python-deadlines:

  • Faça um fork do repositório
  • Edite o arquivo `_data/conferences.yml`
  • Adicione os dados YAML da sua conferência no local apropriado
  • Envie um pull request
3. Formato de Calendário do Python.org

Envie sua conferência para o calendário oficial de eventos Python:

  • Envie um email para href="mailto:events@python.org" >events[at]python.org com o texto no formato de calendário
  • Inclua a linha de assunto: "Python Event Submission: [Nome da sua conferência]"
  • Aguarde a confirmação da equipe de eventos Python
Dica: Envie para as três plataformas para maximizar a visibilidade da sua conferência Python!
