Cara Mengirimkan Informasi Konferensi Anda
1. Format CSV Python-Organizers

Tambahkan konferensi Anda ke repositori python-organizers/conferences:

  • Fork repositori tersebut
  • Edit file CSV tahun yang sesuai
  • Tambahkan baris data konferensi Anda
  • Kirim pull request
2. Format YAML PythonDeadlin.es

Tambahkan konferensi Anda ke jesperdramsch/python-deadlines:

  • Fork repositori tersebut
  • Edit file `_data/conferences.yml`
  • Tambahkan data YAML konferensi Anda di tempat yang sesuai
  • Kirim pull request
3. Format Kalender Python.org

Kirim konferensi Anda ke kalender acara Python resmi:

  • Email ke href="mailto:events@python.org" >events[at]python.org dengan teks format kalender
  • Sertakan subjek email: "Python Event Submission: [Nama Konferensi Anda]"
  • Tunggu konfirmasi dari tim acara Python
Tip: Kirim ke ketiga platform untuk memaksimalkan visibilitas konferensi Python Anda!
