Comment soumettre les informations de votre conférence
1. Format CSV Python-Organizers

Ajoutez votre conférence au dépôt python-organizers/conferences :

  • Créez un fork du dépôt
  • Modifiez le fichier CSV de l'année correspondante
  • Ajoutez la ligne de données de votre conférence
  • Soumettez une demande de pull request
2. Format YAML de PythonDeadlin.es

Ajoutez votre conférence à jesperdramsch/python-deadlines :

  • Créez un fork du dépôt
  • Modifiez le fichier `_data/conferences.yml`
  • Ajoutez les données YAML de votre conférence à l'endroit approprié
  • Soumettez une demande de pull request
3. Format Calendrier de Python.org

Soumettez votre conférence au calendrier officiel des événements Python :

  • Envoyez un email à href="mailto:events@python.org" >events[at]python.org avec le texte au format calendrier
  • Incluez la ligne d'objet : "Python Event Submission: [Nom de votre conférence]"
  • Attendez la confirmation de l'équipe des événements Python
Conseil : Soumettez à ces trois plateformes pour maximiser la visibilité de votre conférence Python !
