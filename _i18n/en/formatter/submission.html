How to Submit Your Conference Information
1. Python-Organizers CSV Format

Add your conference to the python-organizers/conferences repository:

  • Fork the repository
  • Edit the corresponding year's CSV file
  • Add your conference data row
  • Submit a pull request
2. PythonDeadlin.es YAML Format

Add your conference to jesperdramsch/python-deadlines:

  • Fork the repository
  • Edit the `_data/conferences.yml` file
  • Add your conference YAML data at the appropriate place
  • Submit a pull request
3. Python.org Calendar Format

Submit your conference to the official Python events calendar:

  • Email events[at]python.org with the calendar format text
  • Include the subject line: "Python Event Submission: [Your Conference Name]"
  • Wait for confirmation from the Python events team
Tip: Submit to all three platforms to maximize visibility of your Python conference!
