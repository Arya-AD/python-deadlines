So reichen Sie Ihre Konferenzinformationen ein
1. Python-Organizers CSV-Format

Fügen Sie Ihre Konferenz zum python-organizers/conferences Repository hinzu:

  • Erstellen Sie einen Fork des Repositories
  • Bearbeiten Sie die CSV-Datei des entsprechenden Jahres
  • Fügen Sie Ihre Konferenzdatenzeile hinzu
  • Reichen Sie einen Pull Request ein
2. PythonDeadlin.es YAML-Format

Fügen Sie Ihre Konferenz zu jesperdramsch/python-deadlines hinzu:

  • Erstellen Sie einen Fork des Repositories
  • Bearbeiten Sie die Datei `_data/conferences.yml`
  • Fügen Sie Ihre Konferenz-YAML-Daten an der entsprechenden Stelle hinzu
  • Reichen Sie einen Pull Request ein
3. Python.org Kalenderformat

Reichen Sie Ihre Konferenz für den offiziellen Python-Veranstaltungskalender ein:

  • Senden Sie eine E-Mail an href="mailto:events@python.org" >events[at]python.org mit dem Text im Kalenderformat
  • Geben Sie als Betreffzeile an: "Python Event Submission: [Name Ihrer Konferenz]"
  • Warten Sie auf die Bestätigung vom Python-Eventteam
Tipp: Reichen Sie Ihre Konferenz bei allen drei Plattformen ein, um die Sichtbarkeit Ihrer Python-Konferenz zu maximieren!
