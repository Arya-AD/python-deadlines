अपनी सम्मेलन की जानकारी कैसे जमा करें
1. Python-Organizers CSV प्रारूप

अपने सम्मेलन को python-organizers/conferences रिपॉजिटरी में जोड़ें:

  • रिपॉजिटरी का फोर्क करें
  • संबंधित वर्ष की CSV फ़ाइल संपादित करें
  • अपने सम्मेलन का डेटा पंक्ति जोड़ें
  • पुल अनुरोध (pull request) सबमिट करें
2. PythonDeadlin.es YAML प्रारूप

अपने सम्मेलन को jesperdramsch/python-deadlines में जोड़ें:

  • रिपॉजिटरी का फोर्क करें
  • `_data/conferences.yml` फ़ाइल संपादित करें
  • उचित स्थान पर अपने सम्मेलन का YAML डेटा जोड़ें
  • पुल अनुरोध (pull request) सबमिट करें
3. Python.org कैलेंडर प्रारूप

अपने सम्मेलन को आधिकारिक Python इवेंट कैलेंडर में सबमिट करें:

  • कैलेंडर प्रारूप टेक्स्ट के साथ href="mailto:events@python.org" >events[at]python.org पर ईमेल करें
  • विषय पंक्ति शामिल करें: "Python Event Submission: [आपके सम्मेलन का नाम]"
  • Python इवेंट टीम से पुष्टि की प्रतीक्षा करें
सुझाव: अपने Python सम्मेलन की दृश्यता को अधिकतम करने के लिए सभी तीन प्लेटफॉर्म पर सबमिट करें!
